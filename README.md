# Paradise Nursery Shopping Application

Paradise Nursery is an online plant shop built with React and Redux.
Users can browse plants, add them to a cart, update quantities, and view total costs.

## Features
- Browse plants by category
- Add/remove plants from cart
- Increase/decrease quantities
- Dynamic cart total
- Redux state management

## Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Technologies Used
- React
- Redux Toolkit
- React Router
